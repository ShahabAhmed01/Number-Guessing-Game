# 🔮 Number Oracle — Number Guessing Game

A terminal number guessing game written in **C++**.
Pick a number between 1–100. You get **7 attempts**. Can you crack it?

🌐 **Open Source** — Free to use, fork & improve

---

## ⚡ Run
g++ number_guessing_game.cpp -o game
./game

## 🎮 How to Play
- Guess a number between 1 and 100
- Get hints → 🔥 TOO HIGH · 🧊 TOO LOW
- Win by guessing correctly within 7 tries
